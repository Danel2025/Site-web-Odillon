# Logos des Entreprises Gabonaises

Ce dossier contient les logos des entreprises gabonaises qui apparaissent dans la section "Nos Références".

## Logos requis

Placez les fichiers suivants dans ce dossier :

1. **cdc.png** - Logo de la Caisse des Dépôts et Consignations (CDC)
2. **caistab.png** - Logo de la Caisse de Stabilisation (CAISTAB)
3. **seeg.png** - Logo de la Société d'Énergie et d'Eau du Gabon (SEEG)
4. **uba.png** - Logo de United Bank for Africa (UBA Gabon)
5. **sem.png** - Logo de la Société d'Economie Mixte (SEM)
6. **edg.png** - Logo d'Energie du Gabon (EDG)

## Où trouver les logos

### Sites officiels des entreprises :
- **SEEG** : https://www.seeg-gabon.com/
- **UBA Gabon** : https://www.ubagabon.com/
- **SEM** : https://www.sem.gouv.ga/
- **EDG** : https://www.edg.ga/

### Autres sources :
- Sites officiels des entreprises
- Départements de communication des entreprises
- Banques d'images professionnelles (avec autorisation)
- Wikimedia Commons (vérifier les droits d'utilisation)

## Spécifications recommandées

- **Format** : PNG avec fond transparent (ou SVG)
- **Taille** : Minimum 200x100px, recommandé 400x200px
- **Résolution** : 72-150 DPI pour le web
- **Couleur** : Logo original (pas de modification de couleur)
- **Fond** : Transparent de préférence, ou blanc

## Note importante

Assurez-vous d'avoir les autorisations nécessaires pour utiliser ces logos, surtout si c'est à des fins commerciales ou publiques. Contactez les départements de communication des entreprises si nécessaire.

## Fallback

Si un logo n'est pas disponible, le système affichera automatiquement les initiales de l'entreprise avec un style cohérent.
